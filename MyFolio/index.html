About Me

Hi I'm Jonathan Gaspado. I'm a web developer. With 1+ intense experience in web development. I am able to build stunning websites from scratch.

I remember when I'm in my 7th Grade and start using this awesome machine called computer and my first used application called paint. It was fun! Drawing whatever you like and I was amazed.

Through curiosity of what this machine can do more. I studied computer science and web development and by that another curiosity started to open. Wow! I can do things through code. I can do things that others can't. I'm amazing! So, I decided to be a pro on this field. After I graduated in college, I felt insecure with my skills, I felt this feeling of waht they called 'imposter syndrome' so I stop coding. But in 2017 of August through youtube videos, my love of coding returned. Thanks to FreeCodeCamp I'm here again. I'm back and nothing can stop me anymore. :)

Now I'm continue learning and exploring of what this machine can do more specially web development. I'm teaching myself and code every single day specializing front end development.
